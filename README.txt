supybot-scherbengericht
~~~~~~~~~~~~~~~~~~~~~~~

Dieses Plugin ermöglicht die bequeme Einberufung eines Scherbengerichts.
Benutzung auf eigene Gefahr. Für Schäden haftet der Verbraucher.

Basiert auf der Idee und der Umsetzung von erlehmann und kaiser_franz:
https://github.com/erlehmann/scherbengericht [citation needed]

Dieses Programm hat das Ziel, die Medienkompetenz der Leser zu
steigern. Gelegentlich packe ich sogar einen handfesten Buffer
Overflow oder eine Format String Vulnerability zwischen die anderen
Codezeilen und schreibe das auch nicht dran.
